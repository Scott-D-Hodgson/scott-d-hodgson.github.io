Raspberry Pi Setup

Background

The Raspberry Pi computer, also known as the RPI, is product of an initiative by the Raspberry Pi Organization. It is a single board computer (SBC), has become the darling of the Maker community, the "leader" in the SBC industry, and the basis for the hardware used to support the projects in this repository. In particular, the graphical user interface (GUI) components of the Smart Clock (smart-clock-ui) and the Smart Mirror (smart-mirror-ui) use the many of the projects in this repository.

Operating System

The typical operating system OS of the RPI is a variant of the Debian OS (which, in itself, is a build of Linux) called Raspbian. This operating system can be downloaded from the Raspberry Pi Organization's downloads section. Depending on the OS of the computer you are working on, there are different methods to prepare the memory card.

Reference

The HTTP daemon runs under the group, www-data, in the Rasbian OS (the same as in Debian). The html files are stored in the folder /var/www/ (which is owned by the root account) and is readable by all users and only writeable by root.

In order for the user pi to write to /var/www it will need have ownership over the folder.

sudo chown -R pi:www-data /var/www

In addition the user and group permissions will need to be set on the folder as well. 

sudo chmod u+rxw,g+rx-w,o-rwx /var/www

Finally any new file that is created in the folder needs to belong to the group www-data.

sudo chmod g+s /var/www

If there were any existing files or folders, the permissions on those will have to be updated as well.

The setting of the permissions for existing files.

chmod u+rw,g+r-xw,o-rwx

The setting of the premissions for existing folders.

chmod u+rwx,g+rx-w,o-rxw
